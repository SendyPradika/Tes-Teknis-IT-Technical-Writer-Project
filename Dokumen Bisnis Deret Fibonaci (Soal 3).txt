**DOKUMEN SPESIFIKASI ALGORITMA DERET FIBONACCI**

**1. TUJUAN BISNIS**
Algoritma ini dirancang untuk menghasilkan deret Fibonacci hingga suku ke-n yang dimasukkan oleh pengguna, dengan aplikasi dalam analisis finansial, pemodelan pertumbuhan, dan perhitungan rasio teknikal.

**2. SPESIFIKASI INPUT/OUTPUT**
- **Input**: Jumlah suku deret (n) berupa bilangan bulat positif
- **Output**: Deret bilangan Fibonacci hingga suku ke-n

**3. KODE ASLI (BAHASA C)**
```c
#include <stdio.h>

int fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

int main() {
    int n;
    printf("Masukkan jumlah deret Fibonacci: ");
    scanf("%d", &n);
    printf("Deret Fibonacci hingga %d adalah:\n", n);
    for (int i = 0; i < n; i++) {
        printf("%d ", fibonacci(i));
    }
    return 0;
}
